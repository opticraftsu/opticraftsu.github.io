Как играть если при заходе в игру уведомление?

В правом верхнем углу будет кнопка "Play", нажимая на её вы переноситесь в игровое меню.

Как играть по сети в игру?

Подключение к игре друга > Radmine VPN

Можно попробываеть поиграть через Radmine, так как эта процедура занимает очень мало времени.

Создать сеть или присоединиться к сети если твой друг создал её.

Друг создает мир и открывает мир для сети, да на этой версии открытие мира для сети нужна.

А чтобы другой подключился, есть отдельный пункт. Под вкладкой "Показать мои миры".

Присоединится к игре/миру, там вводите айпи адресс сети RadMine, напоминаю не свой айпи а друга.

Как установить скин себе?

Это ксожелению не совсем просто. По поводу установки скинов, нужно скачивать пак скином "SkinPack"

Можно свой создать с помощью программ для пк скинов или через сайтов.

После создания перенесите по пути - C:\Users\Пользователь\AppData\Roaming\OptiCraft\games\com.mojang\skin_packs

И всё, теперь можно в своей игре установить скин
