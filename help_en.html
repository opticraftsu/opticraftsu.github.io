How to play if I get a notification when I enter the game?

In the upper right corner there will be a "Play" button, clicking on it will take you to the game menu.

How to play online?

Connecting to a friend's game > Radmin VPN

You can try to play through Radmin, since this procedure takes very little time.

Create a network or join a network if your friend created one.

A friend creates a world and opens the world to the network, but in this version, opening the world to the network is needed.

And for someone else to connect, there is a separate point. Under the “Show My Worlds” tab.

Join the game/world, enter the IP address of the RadMine network, I remind you not your IP but a friend.

How to install a skin for yourself?

This is unfortunately not entirely simple. Regarding installing skins, you need to download the skin pack "SkinPack""

You can create your own using programs for PC skins or via websites.

Once created, move along the path - C:\Users\User\AppData\Roaming\OptiCraft\games\com.mojang\skin_packs

And that’s it, now you can install the skin in your game
